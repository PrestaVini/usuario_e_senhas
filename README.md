# usuario_e_senhas

Esse projeto associa cada um dos três usuarios disponíveis à uma senha.
Primeiro se tem o input do usuário (caso não seja um dos no hashmap da linha 8, o usuário não é encontrado), depois a senha do respectivo usuário digitado (caso não corresponda a senha do usuário, o acesso não é liberado e se perde uma das 3 tentativas de senha, se essas tentativas se esgotarem o acesso fica bloqueado. Caso a senha seja correta dentro dessas 3 tentativas, o acesso é liberado).

## Tecnologias
- Python

## Como executar
Abra o arquivo com o Visual Studio Code ou app semelhante e execute.
